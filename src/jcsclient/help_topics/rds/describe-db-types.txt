
NAME :
    describe-db-types 


DESCRIPTION :
    Lists the detailed information of a particular database engine supported if a database type identifier is specified.
    Otherwise, RDS lists the information of all the database engines supported.

SYNOPSIS :
    describe-db-types 
    [--db-type-id] <string>


OPTIONS :
    [] => optional parameters

    [--db-instance-id] <string> 
        The unique identifier of the DB Engine ex mysql / postgresql.


EXAMPLES :
    jcs rds describe-db-types
    jcs rds describe-db-types --db-type-id mysql
